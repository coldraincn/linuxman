# 备份与恢复

## 备份

```shell
dump [选项] 备份后文件名 原文件或目录
```

* -level:备份级别
* -f 文件名：指定备份后的文件名
* -u：把备份时间记录在etc/dumpdates
* -v:显示更多信息
* -j：压缩
* -W:显示允许被dump的粪污的备份等级以及备份时间

## 恢复

```shell
restore [模式选项] [选项]
```

模式选项： 

* -C:比较备份数据和实际数据的变化
* -i:进入交互模式，手工选择需要恢复的文件
* -t：查看模式，用于查看备份文件中拥有的数据
* -r：还原模式，用于数据还原

选项： 

* -f：指定备份文件的文件名
