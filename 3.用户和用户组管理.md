# 用户和用户组管理

## y用户配置文件

### 用户信息文件

`/etc/passwd`

### 影子文件

`etc/shadow`

### 组信息文件

`etc/group`

### 组密码文件

`/etc/gshadow`

## 命令

### useradd

```shell
useradd [选项] 用户名
```

* -u UID
* -d home目录
* -c 用户说明
* -g 初始组名
* -G 附加组
* -s shell

### passwd

```shell
passwd [选项] 用户名
```

* -S 查询用户密码状态
* -l 暂时锁定用户，仅root用户可用
* -u 解锁用户

### usermod

```shell
usermod [选项] 用户名
```

* -u UID
* -c 用户说明
* -G 修改附属组
* -L 锁定用户
* -U 解锁

### userdel

```shell
userdel [-r] 用户名
```

* -r 同时删除home目录

### su

```shell
su [选项] 用户名
```

* “-”用户环境变量一起切换
* -c 仅执行一次命令，不切换用户
